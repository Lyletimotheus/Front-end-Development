# Bootstrap 4 Cheatsheet

## Containers
- Wrap around content to contain it
- Help us center our content on the page
- can be FLUID (100% of the vw) or FIXED (varies based on the various breaking points)

# Breakpoints
- min-width: 576px
- min-width: 768px
- min-width: 992px
- min-width: 1200px

# Grid system
- STEP 1: container div surrounding the content
- STEP 2: row div with the content 

# Flex System
- You can specify one of the columns and bootstrap can work out the remaining space the other elements need to occupy.
