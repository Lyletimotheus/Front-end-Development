--------------
CSS Units
--------------
Relative 
    1. EM: with font-size, 1em equals the font-size of the parent. 2em's is twice the font-size of the parent, etc.
            with other properties, 1em is equal to the computed font-size of the element itself. 
    2. REM: relative to the root html elements font-size. If the root font-size is 20px, 1 rem will always be 20px,
            2rem will always be 40px. 
    3. VH
    4. VW
    5. % : is a percentage of the parent element 
            * When it comes to line height it is 50% of the font-size of the element

Absolute
    1. PX
    2. PT
    3. CM
    4. IN
    5. MM
