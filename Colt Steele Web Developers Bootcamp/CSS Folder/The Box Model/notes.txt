The Box Model
--------------
1. Border
2. Padding: space between the actual content and the box border
3. Margin: space outside an elements border between another element

1. Border Properties 
    1.1 BORDER-WIDTH: controls the thickness of the border.
    1.2 BORDER-COLOR: controls the ... color of the border 
    1.3 BORDER-STYLE: Controls the line style - dashed, solid, etc
