1. General Notes on Bootstrap

  • The container class is used to center the elements

2. Alignment

  • Alignment of text uses the class text-(center,left,right,justify,nowrap)
  • Floats are used to quickly align divs to the left or right
  • Aligning blockquote's is simple: class of - "blockquote" or "blockquote-reverse"

3. Lists

  • Removing bullets from the li's just add a class of list-unstyled to the ul
  • Good for menus are inline list and comes with the class of list-inline that needs to be added to the ul

4. Forms

  • You want to group your labels and input in a div with a class of form-control
  • You want to wrap your inputs/ fields in a form-group class

5. Tables

  • Add a class of table to the table tag and everything will line up

6. Grid

  • Is broken up into 12 blocks
